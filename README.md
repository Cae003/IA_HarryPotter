# IA_HarryPotter
Detector de rostos que identifica os personagens em um trecho do filme Harry Potter e a Pedra Filosofal

### Baixar e configurar Microsoft Visual Studio
Para que o projeto funcione corretamente, é necessário utilizar o ambiente de desenvolvimento Microsoft Visual Studio. Rodar no VS Code exige configurações adicionais e mais complexas.

Primeiro instale o Microsoft Visual Studio e configure o ambiente com a seguinte opção:
![Guia para qual ambiente baixar](./assets/Guia.png)

Após isso instale as bibliotecas que estão no "requirements.txt" e pronto! Pode rodar normalmente os códigos!
